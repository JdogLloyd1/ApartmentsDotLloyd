# Alewife Backend

FastAPI + SQLModel backend for the Alewife Apartment Intelligence dashboard.

Full documentation lives at [App V1 Dynamic/README.md](../README.md). The finalized version of this file is produced in Sprint 9 of the [development plan](../../DEVELOPMENT_PLAN.md).

## Quick reference

- Run: `make run` from the repo root (or `./make.ps1 run` on Windows without GNU make).
- Test: `make test`.
- Lint: `make lint`.

Environment: copy [`.env.example`](.env.example) to `.env` and populate secrets.
